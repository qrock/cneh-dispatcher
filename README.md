# Диспетчер цеха

Система управления задачами производства.

## Функционал
- Создание задач с указанием срочности
- Назначение исполнителей
- Отслеживание статуса выполнения
- PWA (установка на телефон)

## Технологии
- Python + FastAPI
- SQLite
- HTML/CSS/JS

## Установка
```bash
pip install -r requirements.txt
python server.py
